Plugin-Powerpoint
Plugin PowerPoint ini memiliki 2 yaitu:

Speech to Text Speech to text ini menggunakan library Speech Recognition yang dimana saya menggunakan google recognition untuk melakukan recognizer untuk lebih jelasnya dapat dilihat pada link berikut ini: https://pypi.org/project/SpeechRecognition/

Word Erro Rate Word Error Rate ini menggunakan Algorithma Levenshtein distance untuk melihat bagaimana cara hitung manual Word Error Rate ini dapat dilihat link berikut ini: https://www.rev.ai/blog/how-to-calculate-word-error-rate/ dan untuk mengetahui lebih lanjut tentang Word Error rate ini dapat dilihat di link berikut ini: https://martin-thoma.com/word-error-rate-calculation/
